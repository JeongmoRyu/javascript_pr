# array helper methods

```jsx
`map`
<!-- 배열에 있는 모든 요소들을 변경하여 새로운 배열을 쓸 때 사용 할 수 있다. 콜백 함수를 이용해 각각의 요소에 호출해서 그 값들을 반화시켜 준다.  -->

`filter`
<!-- 주어진 함수의 기준을 통과하는 모든 요소를 모아 새로운 배열로 반환시켜준다. -->

`find`
<!-- 주어진 함수를 만족하는 가장 앞의 요소의 값을 반환시켜준다. 만약 없다면 undefinded를 반환한다. -->

`every`
<!-- 배열 안의 모든 요소가 함수의 기준을 만족하는지 확인한다. Boolean의 값으로 반환된다. -->

`some`
<!-- 어떤 요소라도, 한 개의 요소라도 주어진 기준을 만족하는지 확인한다. Boolean의 값으로 반환된다.  -->

`reduce`
<!-- 각 배열의 각 요소에 대해서 reduce함수(새로운 함수)를 실행하고 하나의 값으로 반환한다.  -->
```
